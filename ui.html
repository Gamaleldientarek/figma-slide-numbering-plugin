Slide Numbering
Quick start guide
1

Add a number layer

In each slide frame, create a Text layer named {p#}. The plugin replaces its text with the page number.

Slides without this layer are skipped — perfect for covers.

2

Pick a section & run

Choose the Figma Section that holds your slides, then hit Number Slides.

3

Reading order

Slides are numbered left → right, then top → bottom:

123 456 789
